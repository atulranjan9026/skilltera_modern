# SkillTera Modern

Assessment platform frontend.

## Setup

```bash
npm install
```

## Proctoring (face-api.js)

For AI proctoring during assessments, download face-api.js models:

```bash
npm run download-models
```

Models are saved to `public/models/` (~10MB). Run this before first use if models are not committed.

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
